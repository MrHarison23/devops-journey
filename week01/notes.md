# Week 01 — Linux + SSH

## Что изучил

* Что такое Linux
* Что такое SSH
* Как подключаться к серверу
* Как создать SSH-ключ
* Как подключить GitHub через SSH

## Полезные команды

```bash
pwd
ls
cd
mkdir
touch
nano
ssh
git clone
git add
git commit
git push
```

## Что значат команды

* pwd — показать текущую папку
* ls — показать файлы
* cd — перейти в папку
* mkdir — создать папку
* touch — создать файл

## Ошибки, которые встретил

* not a git repository
* Permission denied (publickey)

## Как решил

* вошёл в правильную папку
* создал SSH-ключ и добавил его в GitHub
